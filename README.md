# OpenCart Automation Framework

## Overview
End-to-end automation framework for the OpenCart e-commerce application using Selenium WebDriver with Java.  
Built to automate key workflows, generate detailed test reports, and integrate with CI/CD pipelines using Jenkins and Docker.

---

## Tech Stack
- **Programming Language:** Java  
- **Automation:** Selenium WebDriver  
- **Test Framework:** TestNG (Data-Driven Testing, Grouping, Parallel Execution)  
- **Build & Dependency Management:** Maven  
- **Version Control:** Git & GitHub  
- **Reporting:** Extent Reports  
- **Logging:** Log4j  
- **CI/CD:** Jenkins  
- **Containerization:** Docker  
- **Excel Handling:** Apache POI  

---

## Features Automated
- User Registration & Login  
- Data-Driven Login Testing (valid/invalid scenarios)  
- Product Selection & Add-to-Cart  
- Checkout & Order Confirmation  
- Verification of cart and order summaries  
- Execution of tests in headless and remote environments
